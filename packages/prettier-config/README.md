# @thedaviddias/prettier-config

Prettier config for my personal projects.

## Usage

**Install**:

```bash
$ npm install --dev @thedaviddias/prettier-config

$ yarn add --dev @thedaviddias/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@thedaviddias/prettier-config"
}
```
